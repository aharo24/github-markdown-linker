
``` gio 
For n weights (slopes): y = w_0 + w_1x_1 + w_2x_2 + ... + w_(n - 1)x_(n - 1)
Which, simplified to matrix multiplication is:
y = X * W^T

where X = [x_1, x_2, ..., x_(n - 1)] and W = [w_0, w_1,  w_2, ..., w_(n - 1)]
W^T means W transposed
```
